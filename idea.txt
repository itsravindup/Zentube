# 🧘 ZenTube — Chrome Extension Full Specification

## 📌 Overview

ZenTube is a Chrome Extension (Manifest V3) designed to transform YouTube into a distraction-free, productivity-focused platform. It removes addictive UI elements, provides user-controlled filtering, and promotes mindful content consumption.

---

## 🎯 Core Objective

Reduce distractions on YouTube by:

* Removing algorithm-driven content
* Limiting visual triggers
* Giving users full control over what is visible

---

## 🧱 Tech Stack

* Chrome Extension (Manifest V3)
* JavaScript (Vanilla)
* HTML + CSS (Popup UI)
* Chrome Storage API (`chrome.storage.sync`)
* DOM Manipulation (Content Scripts)
* MutationObserver (for dynamic updates)

## ⚙️ Manifest Configuration

* manifest_version: 3
* permissions:

  * storage
* host_permissions:

  * https://www.youtube.com/*
* content_scripts:

  * Runs on YouTube pages
* action:

  * popup.html as UI

---

## 🧠 Settings Schema

All user preferences are stored in chrome.storage.sync as:

{
"focusMode": true,
"hideHomepage": true,
"hideSidebarWatch": true,
"hideSidebarAll": false,
"hideShorts": true,
"hideComments": true,
"blurThumbnails": false,
"timeTracker": true,
"blockedKeywords": [],
"blockedChannels": []
}

## 🧩 Core Features

### 1. Focus Mode (Master Control)

* Enables/disables all filtering features
* If disabled, extension does nothing

---

### 2. Hide Homepage Feed

* Target: YouTube homepage grid
* Remove:

  * ytd-rich-grid-renderer
* Purpose: Eliminate algorithmic recommendations

---

### 3. Hide Sidebar Recommendations

* Target: #related
* Two modes:

  * Watch-only mode
  * Global hide mode

---

### 4. Hide Shorts

* Targets:

  * ytd-reel-shelf-renderer
  * links containing "/shorts/"
* Remove Shorts from homepage and search

---

### 5. Hide Comments

* Target:

  * #comments
* Remove entire comments section

---

### 6. Blur Thumbnails

* Target:

  * ytd-thumbnail img
* Apply CSS:
  filter: blur(8px)

---

### 7. Time Tracker

* Track time spent on YouTube
* Store:

  * daily usage
  * session duration
* Use background timing logic or tab activity detection

---

### 8. Keyword Blocking

* Hide videos where title contains blocked keywords

---

### 9. Channel Blocking

* Hide videos from selected channels

---

### 10. Popup Control Panel

UI includes:

* Toggle switches:

  * Focus Mode
  * Hide Homepage
  * Hide Sidebar
  * Hide Shorts
  * Hide Comments
  * Blur Thumbnails
  * Time Tracker

* Save settings instantly

* Show “Settings Updated” message

---

## 🔄 Content Script Behavior

* Load settings from storage
* If focusMode = false → exit
* Apply features based on toggles
* Use MutationObserver to reapply changes dynamically
